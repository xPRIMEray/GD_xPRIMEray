# Validation Framework

The system includes structured validation modes designed to quantify ray stability, curvature accuracy, and intersection integrity.

---

## Validation Modes

- Straight-line baseline comparison
- Known GRIN analytical solution comparison
- Energy conservation checks
- Path divergence monitoring

## Performance Metrics

- ms/frame
- Ray steps per pixel
- Convergence tolerance
- Shadow evaluation pruning
