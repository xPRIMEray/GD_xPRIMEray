# Wormhole Dual Reality Story

Short name: **Wormhole DualReality — Storytelling Sequence**

## Purpose

A curated six-step visual narrative explaining the wormhole's `DualRealityTransport` stack to a non-technical audience. Progresses from a clean curved render through reference-reality inset, curvature heat map, semantic glyphs, collision radar, and full-stack overlay — each step adding one interpretive layer.

## What the Output Shows

1. **Clean curved** — Bare wormhole render; no overlays.
2. **Reference Reality** — Straight-path inset for perceptual contrast.
3. **Curvature map** — Heat map of cumulative ray bending.
4. **Curvature + semantic** — Curvature + BLV/portal glyph overlays.
5. **Curvature + collision** — Curvature + collision radar.
6. **Full stack** — All overlays simultaneously.

`storytelling_sequence.md` describes each panel in plain language.
